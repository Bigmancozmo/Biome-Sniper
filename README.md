# <img src="https://raw.githubusercontent.com/Bigmancozmo/Biome-Sniper/refs/heads/main/icon.png" height="24em"> BMC's Biome Sniper <img src="https://raw.githubusercontent.com/Bigmancozmo/Biome-Sniper/refs/heads/main/icon.png" height="24em">
Automatic biome sniper for [Sol's RNG](https://www.roblox.com/games/15532962292/Sols-RNG) with Windows & MacOS compatibility

## Features
- Auto-pause after joining a biome
	- Rare biomes (Glitched, Dreamspace, Cyberspace) bypass this
- Automatic updates
- Customizable Discord servers/channels
- Every biome in the game available

## Usage
- Download the [latest release](https://github.com/Bigmancozmo/Biome-Sniper/releases/latest)
- Run `start-windows.bat` or `start-macos.sh`, depending on your OS
	- for MacOS, you may need to open Terminal and run `chmod +x start-macos.sh`
- Input your Discord token
  - Refer to [this](https://gist.github.com/MarvNC/e601f3603df22f36ebd3102c501116c6) to find it
- Configure the sniper to your liking
